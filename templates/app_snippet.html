QWIKGAME is Installed:
  1. Sign In
  2. Play a Game
Save QWIKGAME:
  1. Open Safari
  2. Tap the Share button (box with an arrow)
  3. Select Add to Home Screen
  4. Tap Add
Enable QWIKGAME push notifications:
  1. Go to Settings
  2. Open Notifications
  3. Select qwikgame
  4. Turn on Allow Notifications
Firefox Extension required:
  1. Please install the
    Progressive Web Apps for Firefox
    extension to enable PWAs.
Browser Support:
  1. Unfortunately this browser does not support PWA's.
  2. The Chrome family of browsers provide good PWA support.
