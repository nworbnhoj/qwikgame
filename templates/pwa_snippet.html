Install QWIKGAME:
  1. Open Safari
  2. Tap the Share button (box with an arrow)
  3. Select Add to Home Screen
  4. Tap Add
Enable QWIKGAME push notifications:
  1. Go to Settings
  2. Open Notifications
  3. Select qwikgame
  4. Turn on Allow Notifications
