{% blocktranslate with QWIK="QWIK" %} {{ QWIK }}GAME is an app which facilitates finding partners and arranging 2-player games. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} {{ QWIK }}GAME focuses on 2-player games only with ambitious plans for multi-player games in the future. Currently we have the most popular 2-player games like tennis, squash, table tennis, badminton, racquetball, but also chess, darts, backgammon, billiards and more. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} {{ QWIK }}GAME is free of charge. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} {{ QWIK }}GAME works anywhere in the world. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} To find a partner and organize a match start by sending an invitation for a match. Select the game, venue and a timeframe. It is recommended to call the venue and check if there are available playgrounds for your preferred timeframe. Your Match Invitation will be visible in the QWIKFeed to other Players. Multiple Players may respond to your Invitation. You get to choose the best offer and confirm the match. You can also reply to an existing invitation from the {{ QWIK }}Feed by placing an offer for a particular hour from the given timeframe. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} You can post as many invitations as you wish every day and for any game, venue or timeframe. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} You can find all current invitations in our {{ QWIK }}Feed. It is contantly updated with new invitations sent by Players. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} The Player who send the initial match invitation is responsible to call and book a court/playground once the match is confirmed. The Player is also recommended to call and check the venue availability before sending the invitation. {% endblocktranslate %}

{% blocktranslate with QWIK="QWIK" %} {{ QWIK }}GAME is a Progressive Web App. It's the new generation of apps. No need to download nor install anything. Just save {{ QWIK }}GAME on your Home screen. {{ QWIK }}GAME PWA works on all operating systems and an Internet Browser is the only requirement. {% endblocktranslate %}
